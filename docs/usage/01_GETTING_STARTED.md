> Step by step to setup and understand the project.

---

## Prerequisites

- Node.js 18+
- npm/yarn/pnpm
- Git


## Environment Setup

```bash
# Clone
git clone <repository-url>
cd <project>

# Dependencies
npm install
# or: yarn / pnpm install
```


## First Steps

1. Clone the repository
2. Install dependencies
3. Configure environment variables (if applicable)
4. Run the project

## Useful Commands

```bash
# Development
npm run dev

# Build
npm run build

# Tests
npm run test
```
